# ICO Stats

## Run your app

**NOTE:**  make sure you are running node 7 or later.

### Terminal #1:

```
npm install
npm run dev
```

### Terminal #2:
```
docker-compose up
```

## Helpers

### npm
```
npm run build   # Create production build
```

### Makefile

Log into container shell
```
$ make shell
```

Run tests
```
$ make test
```
